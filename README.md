# AxerHelper

### Tool to manage axer-bot

## Setup

### Initializing

1. Install Node.js
2. Rename `config.example.json` to `.env`
3. Fill `config.example` with your credentials
4. run `yarn` or `npm i` to install some packages
5. run `yarn dev` to start the bot in dev mode!

### Building

-   Run `tsc`
